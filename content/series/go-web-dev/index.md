# Go Web 后端架构实战

> **写给前端开发者的后端进阶课**
> 
> 你是否厌倦了只写页面和组件？你是否想知道 API 响应背后的逻辑？你是否想拥有独立设计和交付完整产品的能力？
> 
> 本系列文章不只是教你 Go 语法，而是通过从零构建一个**生产级博客系统 (Go-Blog-API)**，带你深入后端的广阔世界。我们将采用企业级标准的**分层架构 (Controller-Service-Repository)**，探讨从数据库建模到高并发优化的每一个关键决策。

## 课程大纲

### 第一阶段：起步与规范
1.  [启程：从前端思维到后端架构](/series/go-web-dev/01.启程：从前端思维到后端架构)
    -   **核心概念**：后端请求生命周期、标准项目目录结构 (Standard Go Project Layout)、配置管理。
    -   **实战**：初始化项目骨架，编写第一个符合规范的 API。
2.  [核心：Gin 框架与 RESTful API 设计](/series/go-web-dev/02.核心：Gin%20框架与%20RESTful%20API%20设计)
    -   **核心概念**：RESTful 资源设计、HTTP 动词与状态码的最佳实践、路由分组策略。
    -   **实战**：设计博客系统的 API 契约，封装统一响应结构。

### 第二阶段：数据与业务
3.  [数据：MySQL 建模与 GORM 最佳实践](/series/go-web-dev/03.数据：MySQL%20建模与%20GORM%20最佳实践)
    -   **核心概念**：关系型数据库范式、索引原理、逻辑外键 vs 物理外键。
    -   **实战**：设计 User、Article、Comment 表结构，配置 GORM 自动迁移。
4.  [架构：分层架构与依赖注入](/series/go-web-dev/04.架构：分层架构与依赖注入)
    -   **核心概念**：为什么需要 Service 层？如何解耦业务逻辑与数据访问？
    -   **实战**：实现用户注册功能，体验 Controller -> Service -> Repository 的调用链路。
5.  [安全：JWT 身份认证与中间件机制](/series/go-web-dev/05.安全：JWT%20身份认证与中间件机制)
    -   **核心概念**：Cookie vs Token、密码哈希 (Bcrypt)、AOP 切面编程。
    -   **实战**：实现登录接口，编写 AuthMiddleware 和全局错误处理中间件。

### 第三阶段：进阶与优化
6.  [事务：复杂查询、事务与分页](/series/go-web-dev/06.事务：复杂查询、事务与分页)
    -   **核心概念**：ACID 特性、N+1 查询问题、游标分页 vs 偏移量分页。
    -   **实战**：实现文章列表（带作者信息）、发布评论（事务保证）。
7.  [性能：Redis 缓存策略与高并发](/series/go-web-dev/07.性能：Redis%20缓存策略与高并发)
    -   **核心概念**：Cache-Aside 模式、缓存穿透/雪崩/击穿、分布式锁。
    -   **实战**：为文章详情页添加缓存，优化热点数据读取。
8.  [异步：RabbitMQ 解耦与后台任务](/series/go-web-dev/08.异步：RabbitMQ%20解耦与后台任务)
    -   **核心概念**：同步 vs 异步、消息队列模型、削峰填谷。
    -   **实战**：注册成功后异步发送欢迎邮件，记录操作日志。

### 第四阶段：工程化与交付
9.  [稳健：单元测试与 Swagger 文档](/series/go-web-dev/09.稳健：单元测试与%20Swagger%20文档)
    -   **核心概念**：测试金字塔、Mock 技术、文档即代码。
    -   **实战**：为 Service 层编写单元测试，自动生成 Swagger 接口文档。
10. [交付：Docker 容器化与云原生部署](/series/go-web-dev/10.交付：Docker%20容器化与云原生部署)
    -   **核心概念**：不可变基础设施、容器编排。
    -   **实战**：编写 Dockerfile 和 docker-compose.yml，一键启动 App + MySQL + Redis。

### 实战项目一：RBAC 权限管理系统
11. [实战一：RBAC 权限系统设计与数据库建模](/series/go-web-dev/11.实战一：RBAC%20权限系统设计与数据库建模)
    -   **核心概念**：RBAC 模型、ER 图设计、GORM 多对多关联。
    -   **实战**：设计 User-Role-Permission 数据库表结构，初始化项目。
12. [实战一：用户认证与 JWT 刷新机制](/series/go-web-dev/12.实战一：用户认证与%20JWT%20刷新机制)
    -   **核心概念**：双令牌机制 (Access + Refresh)、Redis 黑名单、强制下线。
    -   **实战**：实现高安全性的登录认证体系。
13. [实战一：角色与菜单权限管理](/series/go-web-dev/13.实战一：角色与菜单权限管理)
    -   **核心概念**：树形结构算法、递归、关联更新。
    -   **实战**：实现菜单的增删改查，将扁平数据转换为前端需要的 Tree 结构。
14. [实战一：基于 Casbin 的 RBAC 权限控制](/series/go-web-dev/14.实战一：基于%20Casbin%20的%20RBAC%20权限控制)
    -   **核心概念**：Casbin 模型 (PERM)、策略同步、中间件拦截。
    -   **实战**：集成 Casbin 引擎，实现 API 级别的细粒度权限控制。
15. [实战一：前端对接与全链路测试](/series/go-web-dev/15.实战一：前端对接与全链路测试)
    -   **核心概念**：Next.js + Ant Design、Axios 拦截器、Zustand 状态管理。
    -   **实战**：前端视角对接后端 API，实现 JWT 自动刷新和动态菜单渲染。
16. [实战一：生产环境部署与 CI/CD](/series/go-web-dev/16.实战一：生产环境部署与%20CI/CD)
    -   **核心概念**：多阶段构建、Nginx 反向代理、GitHub Actions。
    -   **实战**：搭建自动化流水线，实现代码提交即上线。

### 实战项目二：企业级 B2C 电商平台
17. [实战二：电商核心业务建模与 SPU/SKU 设计](/series/go-web-dev/17.实战二：电商核心业务建模与%20SPU-SKU%20设计)
    -   **核心概念**：SPU/SKU 区分、分类树、商品属性建模。
    -   **实战**：设计商品、SKU、属性、分类表结构，并用 GORM 建模。
18. [实战二：商品列表与详情页的高性能查询](/series/go-web-dev/18.实战二：商品列表与详情页的高性能查询)
    -   **核心概念**：动态条件查询、Redis Cache-Aside、Next.js SSR 数据获取。
    -   **实战**：实现商品列表筛选与详情接口，并通过 Redis 提升读取性能。
19. [实战二：购物车系统与 Redis 高级应用](/series/go-web-dev/19.实战二：购物车系统与%20Redis%20高级应用)
    -   **核心概念**：购物车建模、Redis Hash、未登录与登录购物车合并策略。
    -   **实战**：用 Redis 存储购物车，实现添加、删除、更新、合并等核心能力。
20. [实战二：订单状态机与分布式 ID 生成](/series/go-web-dev/20.实战二：订单状态机与分布式%20ID%20生成)
    -   **核心概念**：订单主表与明细表、订单状态机、Snowflake 分布式 ID。
    -   **实战**：从购物车生成订单，使用事务与分布式 ID 保证一致性与可追踪性。
21. [实战二：高并发库存扣减与 RabbitMQ 削峰填谷](/series/go-web-dev/21.实战二：高并发库存扣减与%20RabbitMQ%20削峰填谷)
    -   **核心概念**：Redis + Lua 原子扣减、消息队列削峰填谷、防止超卖。
    -   **实战**：构建高并发秒杀场景下的库存预扣减与异步下单链路。
22. [实战二：支付回调与全链路联调](/series/go-web-dev/22.实战二：支付回调与全链路联调)
    -   **核心概念**：支付记录建模、回调 Webhook、幂等更新、前后端联调。
    -   **实战**：模拟第三方支付，从下单到支付完成实现完整交易闭环。
